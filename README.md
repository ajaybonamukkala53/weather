🌦️ Weather App

A modern and responsive weather forecasting application built using HTML, CSS, JavaScript, and the OpenWeather API. The application provides real-time weather information for any city, including temperature, humidity, wind speed, atmospheric pressure, and weather conditions through an intuitive and user-friendly interface.

🔗 Live Demo
https://open-weather-2025.netlify.app

🔗 GitHub Repository
https://github.com/ajaybonamukkala53/weather

✨ Features
• Search weather by city name
• Real-time weather updates
• Current temperature display
• Humidity information
• Wind speed details
• Atmospheric pressure
• Weather condition icons
• Responsive design for desktop and mobile
• Clean and interactive user interface
• Fast API integration

🛠️ Tech Stack
• HTML5
• CSS3
• JavaScript (ES6)
• OpenWeather API

1. Clone the repository

2. Navigate to the project

3. Open index.html in your browser or use Live Server.
